--- layout: post title: WordPress 随机日志 最新日志 tags: - php - WordPress - 最新日志 - 随机日志 status: publish type: post published: true meta: _edit_last: '1' _syntaxhighlighter_encoded: '1' --- 我最初调用随机日志是用的wordpress中文工具箱这个插件，这个插件用查询数据库的方式来获取随机日志，但是会出现显示不出来的问题。 在wordpress doc发现wordpress自带有个函数能实现这个功能。 [php] <?php get_posts('arguments'); ?> [/php] [php] //最新日志代码 <?php $rand_post = get_posts('numberposts=5'); foreach( $rand_post as $post ) : ?> <li><a href="<?php the_permalink(); ?>" title="<?php the_title(); ?>"><?php the_title(); ?></a></li> <?php endforeach; ?> //随机日志代码 <?php $rand_post = get_posts('numberposts=5&orderby=rand'); foreach( $rand_post as $post ) : ?> <li><a href="<?php the_permalink(); ?>" title="<?php the_title(); ?>"><?php the_title(); ?></a></li> <?php endforeach; ?> [/php]
