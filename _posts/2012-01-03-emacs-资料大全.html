The Emacs Tutorial

Emacs自带的Tutorial是入门的最佳学习资料。Tutorial是交互的（你可以边看边做）， Built into Emacs itself this should be the first thing you ever read about Emacs. The tutorial is interactive (you get to play with concepts after you read about them) and you can start it at any point (provided you’ve started Emacs first) by pressing C-h t.
