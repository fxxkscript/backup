--- layout: post title: CSS Hack tags: - CSS - hack - ie status: publish type: post published: true meta: _edit_last: '1' _syntaxhighlighter_encoded: '1' --- 可以会用到的一些IE css hack写法，记录一下。 尽量不要用IE条件注释吧。 收集了常用的IE6-9的hack [css] .IE9{property:value\9\0} .IE6-IE8{property:value\9;} .gte-IE-8{property:value\0;} .lte-IE-7{*property:value;} .IE-7{+property:value;} .IE-6{_property:value;} .not-IE{property//:value;} [/css] 还是需要少用hack，其实用hack的机会真的很少。也推荐用高级的选择器来实现hack [css] .selector .child{property:value;} /* for ie-6 */ .selector > .child{property:value;} /* except ie-6 */ [/css]
